# BACKPACK

When you go pairing don't forget to bring your backpack.


Add these plugins to your vim bundle

```
dkarter/backpack
junegunn/fzf
junegunn/fzf.vim
junegunn/vim-peekaboo
terryma/vim-multiple-cursors
scrooloose/nerdtree
dyng/ctrlsf.vim
haya14busa/incsearch.vim
killphi/vim-legend
```

Install fzf from homebrew:

```sh
brew install fzf
```
